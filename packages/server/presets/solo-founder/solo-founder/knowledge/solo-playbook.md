# Solo Founder Playbook

> Step-by-step guide for building a product alone with AI team support.

## Phase 1: Validate (Week 1)
1. **Problem statement** — Write a 1-sentence problem description
2. **Customer interview** — Talk to 5 potential users (even informally)
3. **Competitor scan** — Find 3-5 alternatives, note gaps
4. **Unique angle** — What makes your approach different?

## Phase 2: Build (Week 2-3)
1. **Core feature** — Pick ONE feature that solves the problem
2. **Tech stack** — Choose boring, proven technology
3. **Landing page** — Explain value prop in 3 seconds
4. **Waitlist / signup** — Capture interest before building

## Phase 3: Launch (Week 4)
1. **Beta users** — Get 10 real users from your network
2. **Feedback loop** — Daily check: what's confusing? what's broken?
3. **Iterate** — Fix top 3 complaints, ignore the rest
4. **Public launch** — Product Hunt, Hacker News, Twitter

## Key Principles
- **Ship > perfect** — A working MVP beats a beautiful mockup
- **Talk to users** — Build what they need, not what you imagine
- **One metric** — Pick one number and optimize for it
- **Time-box** — If it takes more than 4 weeks, scope is too big
